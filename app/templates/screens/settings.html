{# ============ ENVIRONMENT INFO ============ #}

Environment Configuration

For server deployments (EC2), set these in your terraform.tfvars or as environment variables. For local development, create a .env file in the project root.

All environment variables 
# App
TRANSCRIBE_STUDIO_HOST=0.0.0.0
TRANSCRIBE_STUDIO_PORT=8082
TRANSCRIBE_STUDIO_DATA=/var/lib/transcribe-studio

# Supabase (alternative to UI settings above)
SUPABASE_URL=https://xxx.supabase.co
SUPABASE_ANON_KEY=eyJ...
SUPABASE_DB_URL=postgresql://...

# S3
S3_BUCKET=your-bucket-name
S3_PREFIX=transcribe-studio
AWS_REGION=us-east-1
AWS_ACCESS_KEY_ID=AKIA...       # not needed on EC2 with IAM role
AWS_SECRET_ACCESS_KEY=...        # not needed on EC2 with IAM role
